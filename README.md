# Azure VM & NGINX Hands-On Project

This repository contains **screenshots documenting my hands-on experience with Microsoft Azure**, where I deployed a Linux virtual machine and configured a live NGINX web server. This project demonstrates practical skills in cloud administration and basic networking.

## Project Overview

The goal of this project was to gain real-world experience in:

- Deploying a Linux Virtual Machine (Ubuntu 24.04) on Azure
- Configuring and managing **Azure Virtual Networks (VNet)** and **Network Security Groups (NSG)**
- Securing the VM using **SSH key authentication**
- Installing and verifying a live **NGINX web server**
- Managing **public/private IPs** and understanding inbound network rules

This project was performed entirely using my Azure free credits, with careful resource cleanup after testing.
## Skills Practiced

- Cloud resource management on Azure  
- Linux system administration basics  
- Web server deployment and verification  
- Network security concepts (SSH, NSG, IP management)  

**This project is intended as a portfolio piece for demonstrating hands-on cloud administration experience.**
